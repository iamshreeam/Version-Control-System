# Version Control System (VCS) 🗂️

**Self Project**  
*July 2026*

## 🎯 Objective
To develop a **C++ CLI Version Control System (VCS)** inspired by Git to manage and restore source code.

## 🛠️ Approach
* **Architecture:** Engineered a **filesystem-based VCS** capable of maintaining a full commit history utilizing a disk-based list and the `C++17 <filesystem>` library.
* **Core Functionality:** Implemented essential VCS operations including `init`, `add`, `commit`, `log`, and `revert` to enable full snapshot creation and management.

## 🚀 Impact
* Achieved comprehensive history management with reliable snapshot creation and restoration capabilities across all standard VCS operations.

## 💻 Technologies & Tools
* **Language:** C++ (C++17)
* **Libraries:** standard `<filesystem>`
* **Interface:** Command Line Interface (CLI)
